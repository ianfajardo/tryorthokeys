## Technical / Structural Issues Spotted

1. **Inconsistent SEO components** — `/olkb` uses raw `<Head>` + `<meta>` tags. Every other page uses `<SEOhead>`. Migrate so structured data and OG tags are uniform.
2. **Keyword stuffing in `/olkb` `<meta name="title">`** — current content is a 2018-style stuffed string ("40% Keyboard Kit ortho keyboard qmk ortholinear keyboard ortholinear keycaps forty percent keyboard Drop planck"). Replace with a single clean title.
3. **Missing `dateModified` updates** — posts that target time-sensitive intent ("2026", "deals", "drop closing") need `dateModified` bumped on every refresh, not just `date`.
4. **No FAQ schema on `/split`** — `/keyboards` and `/keycaps` both have it; `/split` doesn't.
5. **Image alt text gaps** — several `<img>` tags in `/keycaps` have empty `alt=""` (Pimp My Keyboard rows especially). Free win.
